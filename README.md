# CONVERSOR DE TEMPERATURA

![](./img/preview.png)

Este é um aplicativo simples de conversão de temperaturas feito em Java, utilizando a biblioteca **Swing**.  
Ele permite converter temperaturas de **Celsius** para **Kelvin** ou **Fahrenheit**.

## Tecnologias

* Java 21
* Swing
